```
┌────────────────────────────────────────┐
│  InputActionMap / InputAction          │  可重绑定、资源化、阶段机
```

## 10.2 底层门面 — Input

**文件:** `Input.cs`

### Handler 栈

```csharp
Input.PushHandler(IInputHandler);
Input.PopHandler();
Input.Current
```

默认实现 + `NullInputHandler`（无输入环境）。  
编辑器 Play Mode 推入 `GameViewInputHandler`，使游戏输入仅在 Game View 聚焦时生效。

### 键盘

- `GetKey` / `GetKeyDown` / `GetKeyUp`
- `AnyKey`
- `IsCtrl/Shift/AltPressed`
- `GetWASD` / `GetArrowKeys` / `GetKeyCombo`

### 鼠标

- 位置 / delta / 滚轮
- 按键
- 光标形状与可见性

### 光标锁定

```csharp
LockCursor / UnlockCursor
// 锁定上下文栈
OnCursorLocked / OnCursorLockFailed
```

通过 `ICursorLockHandler` 适配平台差异。

### 手柄

按钮、摇杆、扳机、震动。

### Action Map 注册

```csharp
RegisterActionMap / UnregisterActionMap
FindAction / ActionMaps
```

## 10.4 Input Action 系统（Unity 风格）

| 类型 | 角色 |
|------|------|
| `InputActionMap : EngineObject` | 命名动作集；资产扩展名 `.inputactions` |
| `InputAction` | 单动作：绑定 + 阶段机 |
| `InputBinding` | 键 / 鼠标键 / 手柄键 / 轴 |
| `InputCompositeBinding` | 多部件 Composite |
| `InputActionContext` | 回调上下文 |
| `IInputProcessor` | 绑定上的值处理器 |

### 动作类型与阶段

```csharp
enum InputActionType { Value, Button, PassThrough }

enum InputActionPhase {
  Disabled, Waiting, Started, Performed, Canceled
}

enum InputInteractionType {
  Default, Press, Release, Hold, MultiTap, PressAndRelease, Tap
}
```

### InputAction API

```csharp
Enable() / Disable()
AddBinding(KeyCode | MouseButton | GamepadButton | composite)  // 流式
ReadValue<T>()
WasPressedThisFrame / WasReleasedThisFrame / IsPressed
// 事件
Started / Performed / Canceled
```

### Composites（`InputComposites.cs`）

| 类型 | 用途 |
|------|------|
| `Vector2CompositeBinding` | 上/下/左/右 → Float2（可选 normalize） |
| `DualAxisCompositeBinding` | X/Y 双轴 |
| `AxisCompositeBinding` | 正/负半轴 |

典型用例：WASD → 移动向量、D-Pad、扳机轴。

## 10.6 帧内更新

- Action 更新通常在模拟步中使用 **unscaled delta**（暂停时 UI 仍可响应）
- 阶段机按帧推进：Down → Started/Performed，Up → Canceled
- 处理器在读取路径上变换原始值（死区、缩放、取反等）

## 10.7 使用模式

### 代码创建

```csharp
var map = new InputActionMap("Player");
var move = map.AddAction("Move", InputActionType.Value);
move.AddBinding(/* WASD composite */);
map.Enable();
Input.RegisterActionMap(map);

// 每帧
var v = move.ReadValue<Float2>();
```

### 资产驱动

1. 在 Project 中创建 `.inputactions`
2. 在编辑器中配置 Action 与绑定
3. 运行时加载 `AssetRef<InputActionMap>` 并 `Enable` / 注册

### 编辑器注意

- Play Mode 下输入受 Game View 焦点限制
- 编辑器自身快捷键走 `ShortcutManager`（与游戏 Action 分离）
