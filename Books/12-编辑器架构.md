# 12 · 编辑器架构

---

## 12.1 入口与应用壳

| 类 | 路径 | 说明 |
|----|------|------|
| `Program` | `Prowl.Editor/Program.cs` | CLI：`--project`、`--restore-scene`、`--buildmode`、`--output` |
| `EditorApplication : Game` | `Core/EditorApplication.cs` | 单例 `Instance`；完整编辑器生命周期 |

### 与 Runtime Game 的差异

编辑器 **不** 使用 `Game` 默认的场景 Update/Render/GUI 驱动，而是：

| 钩子 | 编辑器行为 |
|------|------------|
| `OnUpdate` | 仅当 `ShouldRunGameplay` 时跑玩法；始终可画 Gizmo |
| `OnRender` / `OnGui` | 空操作；由 `SceneViewPanel` / `GameViewPanel` 负责 |

### UI 栈

- **PaperUI** — Immediate 布局
- **OrigamiUI** — PropertyGrid / 控件
- **DockSpace** — 停靠（默认布局类似 Blender/Unreal：Scene+Game | Project+Console || Hierarchy | Inspector）
- **Rosetta** — 本地化（`Resources/Locale`）

## 12.3 Selection

**文件:** `Core/Selection.cs`（静态）

```csharp
Selected / ActiveObject / Count / OnSelectionChanged
Select / AddToSelection / RemoveFromSelection / ToggleSelection / Clear
HandleListClick(item, allItems, index, ctrl, shift)  // Ctrl 多选, Shift 范围
GetSelected<T>() / GetActiveAs<T>() / IsSelected
```

**Ping 系统：** `Ping(Guid)`、`UpdatePing(dt)`、`GetPingAlpha()` — 临时高亮，不改变选择。

**热重载恢复：** `SelectionToken` / `SelKind`（`Utils/SelectionToken.cs`）按稳定 ID 恢复 GO / Component / Asset / Content。

## 12.4 Undo / Redo

**文件:** `Core/Undo.cs`（静态）

- `MaxSteps = 100`
- **Play Mode 中禁用**

### 记录类型

1. **PropertyRecord** — Echo 前后快照；目标用 `MonoBehaviour.Identifier` 解析（可跨越销毁/重建）
2. **ActionRecord** — 显式 undo/redo 委托
3. **Structural** — 创建/销毁 GO（帧末 flush 时序列化）

### 关键 API

```csharp
Snapshot(target)                    // PropertyGrid 修改前
RegisterAction / RegisterCoalescableAction / RegisterActionGroup
RecordGameObjectChange / ApplyGameObjectChanges
RegisterCreatedObject / RegisterDestroyObject
BeginContinuous / EndContinuous / CancelContinuous  // Gizmo 拖拽合并
PerformUndo / PerformRedo / Clear / IncrementGroup
// 事件
OnUndoRedo
```

PropertyGrid 集成（`EditorApplication.Initialize`）：

```
OnBeginRoot = target => Undo.Snapshot(target)
OnFieldChanged = target => (target as EngineObject)?.OnValidate()
```

## 12.5 Play Mode

```
EnterPlayMode:
  1. Echo 序列化编辑场景
  2. Unload → 反序列化 Play 克隆
  3. Application.IsPlaying = true
  4. 重置 Time 栈，推入 GameViewInputHandler
  5. 完整生命周期加载（OnEnable/Start）

ExitPlayMode:
  1. Unload Play 场景
  2. 恢复编辑器快照
  3. 弹出输入 Handler，恢复时间
```

Play 期间：Undo 关闭；脚本编译结果 **推迟** 到退出 Play 再应用。

## 12.8 项目模型

**文件:** `Projects/Project.cs`

```
ProjectRoot/
  Name.prowl
  Assets/
  Library/   (cache, thumbnails, metadata.db, ScriptAssemblies, ~autosave.scene)
  ProjectSettings/
  Packages/
  Temp/
  Logs/
```
