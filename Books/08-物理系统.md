# 08 · 物理系统

---

## 8.1 概览

- **后端**: [Jitter Physics 2](https://github.com/notgiven688/jitterphysics2)（NuGet `Jitter2`）
- **世界**: 每 `Scene` 一个 `PhysicsWorld`（`Prowl.Runtime/Physics/PhysicsWorld.cs`）
- **组件**: `Prowl.Runtime/Components/Physics/`
- **步进时机**: `Scene.FixedUpdate` 内、组件 `FixedUpdate` 之前

---

## 8.2 PhysicsWorld

### 设置

| 属性 | 说明 |
|------|------|
| `Gravity` | 重力 |
| `SolverIterations` / `RelaxIterations` | 约束求解 |
| `Substep` | 子步进 |
| `AllowSleep` | 休眠 |
| `UseMultithreading` / `ThreadModel` | 多线程 |
| `AutoSyncTransforms` | 自动写回 Transform |
| `EnhancedDeterminism` | 增强确定性 |
| 投机接触 / 持久流形 | 稳定性选项 |

### 事件

`PreStep`、`PostStep`、`PreSubStep`

### 关键 API

| 类别 | 方法 |
|------|------|
| 模拟 | `Update()`、`SyncTransforms()`、`Clear()` |
| 静态体 | `GetOrCreateStaticRigidBody(layer)` |
| 层过滤 | 静态 `IgnoreCollisionBetween` / `EnableCollisionBetween`（经 `LayerFilter`） |
| 网格烘焙 | `BakeMesh(Mesh)` → `BakedPhysicsMesh` |
| 射线 | 多组 `Raycast` 重载（origin/dir、maxDistance、LayerMask、`RaycastHit`） |
| 形状投射 | `ShapeCast` / `SphereCast` / `CapsuleCast` / `BoxCast` / `CylinderCast` / `ConeCast` 及 `*All` |
| 重叠检测 | `Overlap*` / `Check*` |
| 地形 | `RegisterTerrain` / `UnregisterTerrain`（`TerrainHeightmapProxy` + `TerrainCollisionFilter`） |

辅助类型：`RaycastHit`、`ShapeCastHit`、`CollisionMatrix`、`LayerFilter`、`CompositeBroadPhaseFilter`、`CollisionTriangle`。

## 8.3 Rigidbody3D

**文件:** `Components/Physics/Rigidbody3D.cs`（sealed MonoBehaviour）

### 属性

- `MotionType`：Dynamic / Kinematic / Static
- `Mass`、`Friction`、`Restitution`
- 线/角阻尼、重力开关、投机接触、陀螺力、休眠阈值
- 质心、惯性张量、线/角速度

### 力与运动

```csharp
AddForce / AddForceAtPosition / AddTorque
ApplyImpulse / ApplyAngularImpulse
MovePosition / MoveRotation
GetPointVelocity / SetActive
```

### 碰撞事件

```csharp
BeginCollide(Rigidbody3D, ContactInfo)
EndCollide(...)
```

`RigidBodyUserData` 在 Jitter body 上存储组件引用与层。碰撞体自动附着到最近的 `Rigidbody3D`，否则附着到世界对应层的静态体。

## 8.5 约束与关节

**基类:** `PhysicsConstraint` — `ConnectedBody`、`Active`，抽象 create/destroy 对接 Jitter `World`。  
**复合:** `PhysicsJoint : PhysicsConstraint`。

### 原子约束

`BallSocketConstraint`、`DistanceLimitConstraint`、`FixedAngleConstraint`、`HingeAngleConstraint`、`ConeLimitConstraint`、`TwistAngleConstraint`、`PointOnLineConstraint`、`PointOnPlaneConstraint`、`LinearMotorConstraint`、`AngularMotorConstraint`

### 关节

`HingeJoint`、`PrismaticJoint`、`UniversalJoint`

约束在组件启用时创建、禁用/销毁时拆除，并随 `ConnectedBody` 变更重建。

## 8.6 CharacterController

基于形状投射的运动（非完整动态刚体角色）。

| 参数 | 说明 |

## 8.7 WheelCollider

**文件:** `WheelCollider.cs`

射线式悬架 + 轮胎模型，用于载具。

| 类别 | 成员 |
|------|------|
| 驱动输入 | `SteerAngle`、`MotorTorque`、`BrakeTorque` |
| 悬架 | 行程、频率、阻尼比、簧上质量 |
| 摩擦 | 纵向/侧向、抓地饱和 |
| 读出 | `IsGrounded`、`SuspensionCompression`、`AngularVelocity`、`WheelRotation`、接触点/法线 |
| 视觉 | 可选 `visualTransform`、`GetWheelCenter()` |

示例：`Samples/CarPhysicsDemo`。

## 8.8 层与过滤

- 使用 `LayerMask` / 场景层系统（`TagLayerManager`）
- `CollisionMatrix` + `LayerFilter` 控制宽相过滤
- 静态 API 可对特定 body 对 ignore/enable
- 地形使用专用 `TerrainCollisionFilter`

## 8.9 与生命周期的关系

```
Scene.FixedUpdate
  ├─ ComponentRegistry.Start (残留)
  ├─ PhysicsWorld.Update()     ← 求解、事件
  │     └─ (可选) AutoSyncTransforms 写回 GO.Transform
  └─ ComponentRegistry.FixedUpdate  ← 脚本读碰撞结果/施力
```

建议：

- 在 `FixedUpdate` 中施力与读物理状态
- 避免在 `Update` 中混用未同步的变换
- 销毁 Rigidbody 前确保约束/碰撞体已正确拆解（组件 OnDisable 会处理常规路径）
