## 16.2 Samples 说明

每个 Sample 是独立 Exe，模式为 `class XxxGame : Game` + `Run(...)`。

| 示例 | 焦点 |
|------|------|
| **SimpleCube** | 最小网格/渲染启动 |
| **PhysicsCubes** | 刚体与碰撞体堆叠 |
| **ShapeCastDemo** | `PhysicsWorld` 形状投射 |
| **PhysicsTesterDemo** | 更广的物理查询 API |
| **CarPhysicsDemo** | `WheelCollider` 载具 + 第三人称相机 |
| **AudioDemo** | 3D 空间音频 + InputAction + 效果链 |
| **FlyCamera** | 自由飞行相机；含 Marching Cubes |
| **LifecycleTest** | MonoBehaviour 生命周期验证 |
| **VoxelEngine** | 体素/Chunk 风格演示 |
| **BananaMan** | 角色/玩法向综合示例 |

**学习路径建议：**

1. `SimpleCube` → 理解 `Game` 与最小渲染
2. `LifecycleTest` → 组件回调顺序
3. `PhysicsCubes` / `ShapeCastDemo` → 物理
4. `AudioDemo` → 音频 + 输入动作
5. `CarPhysicsDemo` → 复杂组件组合
6. 阅读 Editor 工程 → 完整工具链

## 16.3 无编辑器使用 Runtime

```csharp
public class MyGame : Game
{
    public override void Initialize()
    {
        // 安装玩家资源库、加载场景或纯代码搭建
        var scene = new Scene();
        Scene.Load(scene);
        var go = new GameObject("Camera");
        go.AddComponent<Camera>();
        scene.Add(go);
    }
}

// Program.cs
new MyGame().Run("My Game", 1280, 720);
// 或
new MyGame().RunHeadless(new HeadlessRunOptions { ... });
```

项目引用：

```xml
<ProjectReference Include="path/to/Prowl.Runtime.csproj" />
```

无需引用 `Prowl.Editor`。

## 16.4 扩展点清单

### Runtime

| 扩展点 | 方式 |
|--------|------|
| 自定义组件 | 继承 `MonoBehaviour` |
| 自定义渲染管线 | 继承 `RenderPipeline`，赋给 `Camera.Pipeline` |
| 自定义后处理 | 继承 `ImageEffect`，加入 `Camera.Effects` |
| 自定义着色器 | `.shader` DSL + Material |
| 自定义音频效果 | 实现 `IAudioEffect` |
| 自定义网格特征 | `IMeshFeature` / `MeshFeatureRegistry` |
| 输入 | `InputActionMap` 或底层 `IInputHandler` |
| 资源引用 | `AssetRef<T>` + 玩家 `IAssetDatabase` |

### Editor

| 扩展点 | 方式 |
|--------|------|
| 自定义 Inspector | `CustomEditor` |
| 字段绘制 | `PropertyEditor` |
| 资产编辑器 | `AssetEditors/*` 模式 |
| 场景视图工具 | `ISceneViewEditor` |
| 菜单项 | `[MenuItem]` / `MenuSystem` |
| 导入器 | `[ImporterFor]` + `AssetImporter` |
| 缩略图 | `ThumbnailGenerator` 注册 |
| 启动初始化 | `InitializeOnLoad` 类注册表 |
| 热重载清理 | `IScriptReloadCleanup` / `[OnAssemblyUnload]` |
| 构建 | 扩展 `BuildPipeline` / Profile（高级） |
